# Custom-folder-icon
Choose a custom folder icon from dedicated folder

I created this plugin to quickly replace folder images on the fly. This way you don’t have to manually add an image, rename it to folder, and add a dot to the filename to hide it from the library. Just long‑press on a folder and choose “Folder icon”. It will open the KOReader directory.

I recommend preparing your preferred images (small and lightweight) in your preferred format (SVG/PNG/JPG) inside a dedicated folder. Then simply select the image and apply it. To remove it, repeat the process: long‑press on a folder and choose “Remove icon”.

It’s perfect if you have many folders and like to change their images frequently.
